# smarttv_remote
remote control of the smarttv box from andriod

Приложение для удаленного доступа к телевизору.

Работает исключительно в рамках WiFi без доступа в Интернет.

# Процесс компляции простой:

- В Andriod Studio создаете два проекта Kotlin и раскладываете файлы.
- Либо просто устанавливаете клиента на телефон или планшет, а серверную часть на smarttv приставку.
- Серверная часть ставится как сервис, поэтому привычных интерактивных окон не будет.


# После установки APK на телефон:

- Откройте Настройки -> Специальные возможности.
- Найдите TV Server.
- Разрешите запрос функций через 3 точки.
- На экране появится IP и PIN код для подключения.
- В клиенте укажите эти данные, чтоб подключиться к управлению телевизором.
